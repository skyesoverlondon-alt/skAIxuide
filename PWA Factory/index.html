Strategy Guide
Cache First
Serve from cache, fall back to network. Best for: static sites, icons, fonts, shell assets. Fastest perceived load.
Network First
Try network, fall back to cache. Best for: APIs, dashboards, data that must be fresh. Still works offline.
Stale While Revalidate
Serve cached immediately AND fetch fresh in background. Best for: content sites, news, images. Zero wait + stays current.
Cache Only
Only serve from cache. Perfect for fully offline apps where network calls aren't needed at all.
