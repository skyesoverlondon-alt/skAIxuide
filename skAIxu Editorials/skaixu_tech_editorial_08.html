ZIP safety: zip-slip

Archives can be weaponized.

Reject unsafe paths (../, absolute paths). Validate filenames. Extract to a controlled namespace.

08 • Tech Editorial • Production endpoints closed

Manifests and indexing

Large zips should not freeze the UI.

Index projects after import: manifest lists files, sizes, types, and stable IDs. Load blobs lazily.

08 • Tech Editorial • Production endpoints closed

Deterministic exports

If export changes structure, users lose trust.

Preserve directory structure and encoding. If normalization occurs, do it intentionally and document it.
