How to use DataForge AI

Generate highly realistic mock data, strict TypeScript schemas, or relational SQL databases in seconds using AI.

1
API Setup The environment provides a default key automatically. You can also override it by entering your own Gemini API key in the top right header.
2
Describe Data Write a natural language prompt describing what you need. E.g., "A real estate listing with an array of images, an owner profile, and geo-coordinates."
3
Choose Format Select JSON for raw endpoint data, TypeScript for type interfaces, or SQL for table structures and `INSERT` statements. Adjust the count slider for the number of records needed.
4
Forge & Export Click "Forge Data" to let the AI do the work. Once finished, use the Copy or Download buttons to grab your code.
